# cordova-light
## Summary
Find the GPS position where you are and put every 2 seconds a marker on google map.
vue.js + cordova
![geoloc](http://patricerolland.free.fr/pic/ionic_geoloc.jpg)

## Setup
- install cordova
- install android studio or android sdk
- install jdk1.8
- cordova prepare
- npm install
- cordova build android
- cordova run android
